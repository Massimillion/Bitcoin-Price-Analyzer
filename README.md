# Bitcoin Price Analyzer

A real-time Bitcoin price analysis tool that evaluates short-term 
price momentum and market signals on a 5-minute interval, designed 
to support fast trading decisions.

## What it does
- Pulls live Bitcoin price data on 5-minute intervals
- Analyzes short-term price trends and momentum indicators
- Visualizes price movement to identify patterns
- Designed for traders who need quick, digestible market reads

## Why I built it
Most crypto analysis tools are either too complex or too slow for 
short-term decision making. This tool strips it down to what matters 
on a 5-minute timeframe.

## Tech & Tools
- Built using Perplexity with Claude Code
- Live cryptocurrency price data integration
